# 404 / Error Page

> 找不到頁面。**這是設計師展現個性的好地方**。不要套 lucide-style 「Oops!」illustration。

## Anatomy

- Status code（404 / 500）
- Headline
- 1-2 句解釋 + 建議下一步
- Primary action（回首頁 / 搜尋）
- 可選：search bar / recent posts

## Canonical HTML

```html
<main class="error-page" role="main">
  <p class="error-code" aria-hidden="true">404</p>
  <h1>Page not found</h1>
  <p>The page you're looking for doesn't exist. It may have been moved.</p>
  <nav aria-label="Helpful links">
    <ul>
      <li><a href="/">Go home</a></li>
      <li><a href="/search">Search</a></li>
      <li><a href="/contact">Contact us</a></li>
    </ul>
  </nav>
</main>
```

注意：`<h1>` 是「Page not found」，**不是 logo / "404"**。

## 風格輸出對照

| Style | 形式 |
|-------|------|
| Type-First Monochrome | huge "404" bleed, tiny links below |
| Brutalist | default browser 404 style + plain link |
| Swiss Editorial | numbered "ERROR — 404 — PAGE NOT FOUND" baseline |
| Editorial Maximal | "404" as title + editorial paragraph |
| Magazine Longform | full-bleed photo + editorial copy |
| Fashion / Luxury | full-bleed image + tiny tracked text |
| Aurora UI | 404 with aurora glow（解鎖） |
| Glassmorphism | floating glass card with 404 |
| Bento Grid | bento "404" + grid of helpful links |
| Neobrutalism | 404 with hard shadow + bold |

## Accessibility 必查

- [ ] HTTP status code 真的是 404（不只 visual）
- [ ] `<h1>` 寫「Page not found」描述問題，不是只 "404"
- [ ] `.error-code` 數字 `aria-hidden="true"`（SR 已從 h1 知道）
- [ ] 提供 ≥ 2 個 recovery action（home + search）
- [ ] 不要 auto-redirect

## 500 變體

```html
<main role="main">
  <p aria-hidden="true">500</p>
  <h1>Something went wrong</h1>
  <p>We've been notified and are looking into it. Try refreshing in a few minutes.</p>
  <button onclick="location.reload()">Refresh</button>
  <a href="/">Go home</a>
</main>
```

## AI Tell 自查

- [ ] **不要** "Oops! Page not found" 萬用 copy
- [ ] **不要** SVG astronaut / robot / question-mark illustration
- [ ] **不要** 永遠置中佈局（左對齊更有 personality）
- [ ] 善用 404 展現品牌個性（玩笑話、ASCII art、creative type）
